買了高級鍵盤

Jul 31st, 2010

由於先前把 mac mini 跟鍵盤都拿回家裡給媽媽用了, 所以現在手邊只剩下一台有點不夠力的 macbook, 然後下班繼續寫 code 的狀況維持一陣子, 覺得手真的是快廢了, 後來就覺得 macbook 鍵盤超難打, 所以打算買個高級鍵盤。

而一直以來常在推特上看到小恩恩在講高級鍵盤的事情, 又最近看到了小恩恩不小心代言的高級鍵盤, 所以就想說也來買一支高級鍵盤好了, 畢竟每天從早到晚都要寫大量的 code, 所以詢問推特大神之後, 大家都推薦金鍵盤, 所以 google 一下金鍵盤的地址之後, 下班就跑了過去。 不過有點難找, 尤其我對光華商場本來就不熟 ~”~

到了金鍵盤之後, 覺得金鍵盤的老闆人真的超 nice, 跟我解釋很多東西, 青軸、茶軸、黑軸、白軸 … blah blah blah, 總之試打了一會, 看了價位就買了 Filcon 的 Mejestouch 不知道型號的黑軸鍵盤, 花了 3000 塊。

然後老闆還問說有沒有需要 N-key, 所以又跟我解釋了 N-key 的用途, 然後又問需不需要中文。最後當然是不用 N-key 不用中文, 單純英文鍵盤即可, 看起來乾淨多了~ 總之老闆人超 nice ~

不過其實我覺得最好打的是架上標價 6800 的鍵盤 … 也許下個月就可以買一支放公司用之類的。 畢竟靠打字賺錢的, 對自己的手好一點也是應該的 (笑)
