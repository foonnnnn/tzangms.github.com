部落格有名字囉! 「Oceanic | 人生海海」

Aug 22nd, 2007

我的部落格終於有名字了! 話說起因是昨天在 twitter 上, 跟某位跟我一樣玩 Linux、寫程式的音樂製作人 - 阿怪說到 VCS (版本控制系統) 的東西。 一開始我是在 twitter 上說, 該來好好研究 SVN, 不然程式設計師一多, 程式會很難維護。

阿怪看到就回我說:「水銀!!」噗, 我完全搞不清楚狀況, 原來阿怪說的是 mercurial 這個東西, 跟 SVN 類似也是一個 VCS。接著我就又提到, 先前 linus 不是才搞一個 git ? 媽的! 先前才為了在 PHP 上面選用那一套 framework 才煩惱好一陣子, 選錯的話, 有時候可能會一失足成千古恨阿! XD

然而當選定了我想要用的 PHP framework, 現在又有 VCS 要選嗎? 後來 google 了一下, 這些 VCS 在用途上似乎稍有不同? 阿災, 還沒時間去研究。

Anyway, 總之在當時我就整個感覺很悶, 有種想吐的感覺 (這是稍微誇大了一點啦 XD), 難道我得了資訊焦慮症? 因為不光是 framework 和 VCS, 還有每天都要讀上的大量 RSS feeds 阿!

所以我每天一到公司, 就是強迫自己要先看完這些每天都會冒出來的文章, 幾乎每天都有一百則以上的新文章要讀, 有些很簡短, 有些則是朋友的 blog, 要讀上最久的就是程式設計的文章了, 然而這些關於 web 2.0, 程式設計的文章, 全部都是以英文為主, 這會讓我看比較慢一些。

但是身為”網路”程式設計師, 這應該算是不可避免的一件事, 然而有時候碰到一些要花上一些時間去讀的文章, 我甚至不敢打開來看, 深怕自己會吐出來, 另外因為上班的工作仍然要進行, 所以時間不多… 漸漸我的 bloglines mark 起來先略過不讀的文章真的是越積越多了。所以說我覺得我應該快得了資訊焦慮症 XD

噗~好像離題了, 總之就是因為以上種種的原因, 我就在 twitter 上打了

blog 一直都沒名字, 叫 Information Anxiety 好了…

沒想到小草馬上就回我:

可以叫做「Ocean Life｜人生海海」。。。

看到這個, 我愣住了幾秒, 接著回說:

靠!! 太屌! 就是這個了!! 「Ocean Life｜人生海海」

不過接著因為英文文法還有簡潔有力的關係, 最後小草建議這個名字

其實也不用，直接 Oceanic｜人生海海，簡潔有力。

沒錯! 我的部落格名稱就這麼定了!! 不愧是小草! 乾溫! 這個名字真的是太屌了! (大拇指) 不過我還是部落格敘述用上 Information Anxiety, 嗯~ 多帥阿!! Oceanic 這個字真的是太有 fu 了! 另外”人生海海”也很有力阿!! 總之太帥啦!! 哈哈哈!

突然我也想把我的樂團改名字了 XD
