Fedora 台灣愛用者社群 (FTUG) 贊助活動開跑!

Aug 7th, 2007

好久沒有在我的 Blog 寫到跟 Linux 相關的東西了, 這次是為了 FTUG 的贊助活動而特地寫這一篇的。雖然小弟只在這個社群當過一段短暫時間的怠職板主, 也沒真的參與過這個社群所辦的活動, 原因是我已經轉換跑到到 Mac 好長一段日子了, 所以也沒辦法繼續兼顧 Linux 這一塊了。

說到我玩 Linux 也有好長一段時間, 記得當初還在讀高中的時候, 接觸到一本名叫光碟月刊的雜誌, 當中介紹了許多 Linux 的東西, 還有每一期都會附上一片光碟, 裡面滿滿都是 Linux 的應用軟體, 我就在當中看到某個叫做 XFCE 的 Window manager, 那個畫面到現在還記憶猶新阿!! 當時我就深深的被這個與 Windows 極其不同的畫面所吸引, 對 Linux 產生了極大的興趣。

然而高中那時後, Linux 的資訊極其的少, 上網也是透過 33.6k 的數據機撥接上線, 然而 Linux 又是透過網路來散怖, 所以當時為了要取得 Linux 的套件來安裝真的是很麻煩, 幸好在高職的時候, 讀的是資訊科, 常在電腦中心混, 所以就利用了學術網路來下載當時最新的 Slackware 3.2, 不過當時並不是下載所謂的 iso 檔, 燒成光碟開機就可以開始安裝了, 當初我的電腦還不能用光碟開機勒!! Orz

還記得當初 Linux 開須都需要製作兩張開機片, 分別是 boot.img 跟 kernel.img 這兩個映像檔(如果沒記錯的話), 要用 rawrite.exe 將這兩個磁碟映像檔分別寫到兩張 3.5” 磁片, 然後用磁片開機, 只記得當初真的滿坎坷的, 我下載到的 Slackware 3.2 似乎不是完整的光碟版本, 所以我還要指定安裝檔案位置, 不像現在那麼方便就可以安裝。

在好不容易在裝完之後, 還是不會用, 更不用說 X Window 了, 畢竟是高職才開始接觸電腦, 當初一開始就是玩 Windows 95, 也不會 DOS, 所以電腦開機進了 Linux 之後, 那一片漆黑的 console 畫面讓我不知所措, 我只能按電腦面板上的 Power 按鈕來關機….

然而在高三的時候, Redhat 出現了, 雖然當時仍然是 console 的安裝畫面, 但是他的出現讓許多人驚訝他的安裝竟是那麼簡單, 慢慢的雖著 RedHat 的版本更新, 也開始走向 X-Window 的安裝界面, 讓 Linux 更容易上手, 加上 Desktop Enviroment (KDE、GNOME) 的開始起步, RedHat 讓 Linux 整個紅了起來, 當時在 RedHat 7.2 就開始有所謂的取代 Windows 桌面的喊話 XD 然而 RedHat 的 RPM 更是一絕, 讓軟體的安裝變得超簡單。

不過到了讀二專時, 由於沒有網路可用, 所以在當時的山水蘭陽 BBS 上有認識一位叫 YHH 的 Linux 前輩, 他熱心的燒了一片 RedHt 7.2 給我, 當時真的是痛哭流涕, 不過回到學校裝完之後, 也因為沒有網路, 根本就作不了什麼事, 所以二專這段時間, 根本就是沒有在玩 Linux, 一直到上了二技之後碰到了班上的 frank 才又開始燃起 Linux 的希望, 當時 frank 已經是 RHCE 了, 超酷的! 所以就又開始玩起了 Linux! 跟 frank 一起討論玩 Linux 的樂趣。

由於當時二技用的是一台超爛的筆記型電腦, 不知道為什麼 Slackware 8 跟 9 都裝不起來, 只有 Redhat 8 可以… 但是當時已經出到 RedHat 9 了, 而且基本上我不是很喜歡 RPM 的套件管理, 相依性的問題有點麻煩, 當時也並沒有 yum, 所以就在網路上找了一些其他的 Linux distro, 也忘記為什麼會接觸到 Gentoo Linux, 然後就深深被 Gentoo Linux 的理念跟套件管理給吸引住了, 不過最重要的一點就是! Gentoo Linux 在我那一台爛電腦可以裝的起來阿!!!

Gentoo Linux 可以玩得東西可多的勒! 然而 Linux 要玩什麼? 最主要就是系統調較囉! Gentoo Linux 自由度又高… 畢竟我不喜歡龐大的 DE (Desktop Enviroment) 像是 KDE 或是 GNOME, 甚至是 lightweight 一點的 XFCE, 到現在我還是最喜歡 WindowMaker。所以由於某種原因, 讓我把 Windows 整個砍了, 電腦只剩下 Gentoo Linux, 完全不用 Windows 了。 不過在大四的時候, 由於有錄音的考量, 但是當時我覺得我不可能在用 Windows 了, 但是 Linux 在這方面的支援又不夠, 所以我毅然決然的跳槽, 買了一台 Apple 的 iBook, 從此之後, 我開始跳進了 Mac 的世界了, 不過我還是在這台 iBook 上裝了 PPC 板的 Gentoo Linux, 由此可看出我對 Linux 的熱愛 XD

說了那麼多, 然而我對 Linux community 上的貢獻只有那麼一些些, 像是在高三由於熱愛 Linux 跟 starcraft, 還在 linuxfab 發表了一則專欄文章, 教大家如何在 Linux 上用 Wine 來玩 Starcraft, 寫 script 直接跑 sc, 或者是在 BBS 或是 Gentoo forum 多少回覆一些文章, 一直到前幾天 FTUG 的 pake 大大找上我, 問我能不能捐款, 贊助一下 FTUG 的運作, 當然是一口氣答應阿! 我捐了 $999 , 為什麼不捐 $1000 呢? 因為我覺得 $999 比較帥! XD 我想這是我目前能為當初熱愛的 Linux 做出的一點貢獻吧!

最後的重點是, FTUG 贊助活動開跑了!! 詳情請看活動網頁, 對 Linux 有熱血的朋友們, 衝阿!!
