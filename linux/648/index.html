Ubuntu Linux on iBook

May 27th, 2006

前天在我的iBook上裝了Ubuntu Linux 5.10，第一件事就是居然Ubuntu在安裝的時候console居然可以顯示中文，真的是太帥氣了，真想知道怎麼辦到的。很早以前要在console顯示中文要裝Chdrv，後來有jmcce、zhcon跟unicon，不過感覺都沒有跟系統作一個很棒的結合，所以當在安裝ubuntu的時候，整個興奮!!雖然在安裝完之後，中文字變成了方塊，但是我想把中文弄好應該不會有啥問題了!而且應該滿好玩的!

目前正在學apt-get，雖然先前的Fedora還沒學好XD 但是Frank一直跟我推薦ubuntu，所以就裝裝看，雖然還是沒有package database，這感覺很不方便，因為怎麼知道，有哪些package可以裝? 那package在系統上面的名稱又是什麼?不像在Gentoo把幾乎所有網路上Linux的套件都分門別類作成一個套件資料庫，在網路上也有一個獨立的資料庫網頁，在系統上也有分類目錄可以去查看，這樣一來就知道在Linux上有什麼套件可以安裝，是什麼類型的，方便許多，因為當你不知道要看電影要裝什麼套件的時候，直接到media-video目錄看有哪些東西，就可以知道軟體名稱跟描述，然後直接打一行指令就可以安裝，不然連套件名稱都不知道怎麼apt-cache search?? 還要上論壇或者是BBS問說，看電影要裝啥軟體?

然後在裝scim的時候，ubuntu的新手指南雖然寫的很清楚，但是單單裝一個scim就要apt-get好幾個套件，難道不能直接apt-get install scim然後直接就把scim跟其他必要性的套件裝到好嗎?雖然不是很清楚apt-get的整個套件系統，但是新手指南都這樣寫了，就感覺是不是還有相依性的問題? 感覺yum跟apt的套件系統，就像是一個未完成的 Gentoo portage? 但是訴求也不一樣，而且gentoo的套件系統是用編譯起來的，所以也不能拿來跟binary system一起比較，只是長久使用Gentoo下來，排除要花費長時間來編譯所要安裝的套件之外，Gentoo的套件系統是比較貼心跟方便的呢!

Anyway，我在繼續摸Ubuntu跟他的套件系統吧!
