在Mac OS X 上啟動 Apache 的 PHP 支援

Jan 26th, 2006 | Comments

Mac OS X Tiger 基本上就內建了Apache 2跟PHP 4了，所以說，只要改一下設定檔就可以使用PHP了，非常方便。

第一個步驟就是修改Apache的設定檔囉。在這個地方我是開啟Terminal跟vi來做示範。

一開始就是開始修改Apache的設定檔了，由於此設定檔需要管理者權限才能修改，所以在開啟vi修改的時候需要加上sudo來使用管理者權限，在鍵入下列這一樣指令後，系統會要求輸入管理者密碼。

sudo vi /etc/httpd/httpd.conf

再來就是編輯Apache的設定檔了！只有兩個地方要修改。一進入vi就搜尋php這個字，就可以找到第一點，如下圖反白的那一行，只要把這一行最前頭的井字號( # )拿掉就可以了。(下圖為修改後)
