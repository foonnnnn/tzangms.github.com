Safari 3.1 沒說的功能?

Mar 31st, 2008

在用 Safari 的時候, 點連結的預設動作是開新視窗, 但是明明有 tab 功能, 怎麼不開新標籤頁呢? 幸好在 Safari 3.1 可以將連結的預設動作改為開啟新標籤頁,  雖然現在還是隱藏的XD anyway, 只要打開終端機(在應用程式的工具程式資料夾裡), 然後在終端機輸入下列指令即可:

defaults write com.apple.Safari TargetedClicksCreateTabs -bool true

其實這功能早該出來了吧!? 尤其是在 Mac 上的 Command + tab, 只能用來切換”程式”, 如果像是 Safari 有兩個視窗的話, 根本切換不到被蓋住的 Safari 視窗阿! 所以將開啟連結的動作, 改為預設在新標籤頁開啟, 整個方便了許多!

另外~ Safari 沒有開新分頁的按鈕(除非裝外掛), 這讓我很頭痛, 因為通常都用滑鼠在瀏覽網頁, 要開新的標籤頁的話, 還要按 Command + T (因為後來重灌後懶得裝外掛了), 沒想到現在 Safari 3.1 在標籤頁列上面, 可以直接在空的地方滑鼠連點兩下, 就可以開新 tab 了勒!! 這兩個新功能真的是很方便阿!!
