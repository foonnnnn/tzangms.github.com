Garageband Workshop 小記

Feb 8th, 2009

2/7 辦的 GarageBand Workshop 應該還 ok 吧!? 也許來參加的朋友們, 有些人覺得只是 loops 拼拼貼貼, 做出來的東西也沒什麼, 但是我倒是覺得滿多人做出來的音樂都還不錯。 其實該這麼想, 你已經做出音樂了阿!!! 不是嗎?

這次首先要感謝 randylien 大大, 帶了 Macbook 的視訊轉接線, 否則整場就挫賽了! 我什麼都有帶, 就忘記了要輸出到投影機的轉接線 Orz 自己帶了一條音源線, 還帶了錄音界面, 也帶了 remote 來控制 keynote 播放, 就是忘了轉接線… 出門還一再檢查有沒有東西忘了帶 Orz

而這次投影片跟 live demo 講了 1 個多小時, 把大部分的功能都有講到, 只是還稍微遺漏了一點東西, 在大家在實作的時候有人發問才又講一下, 不過基本上, 關於 garageband 的界面, 軟體樂器的概念, 以及 loops 操作, 以及一些快速鍵都有提到。 應該是 GarageBand 真的很好懂, 所以大家在實作時也沒有太多問題。

不過關於場地的部份, 在音樂播放的時候, 效果並不太好, 這點以後得找一些比較好的場地, 或者就是要包整個場地, 直接用店裡的音響會比較好。 另外每個人都要帶電腦來的話, 昨天定的場地是稍微小了那麼一些些 :p

而昨天比較多的意見則是, garageband 內建的 loops 元素太少, 不過我想在這次的 workshop 之後, 大家就可以學到如何操作 GarageBand 跟一些觀念, 以後在去取得 Jam Packs 或是其他廠商做的 loops 來擴充。

最後感謝大家的參與, 如果以後有機會在舉辦類似的活動的話, 也希望大家多來參與, 有什麼意見的話, 也歡迎跟我說嘿。 另外也希望大家能夠有更多的 response XD

希望大家都有學到東西! (老師貌), 另外這次是第一次自己辦活動, 其實還滿好玩的 XD

當天的照片 ( 其實沒拍幾張 ) 可以到我的 flickr 看看。

另外這裡是當天的投影片 ( keynote ‘09 )
