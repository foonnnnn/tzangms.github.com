My Own Dashboard Widget: Email Notifier

Mar 12th, 2007

這兩天因為值班, 所以都待在家裡, 也不知道為什麼就開始寫 Dashboard widget …..明明還有好多東西要讀。只記得是想寫一個可以用來看有多少 unread mail 的東西, 雖然這網路上, 似乎已經有其他類似的東西了, 不過我一開始好像是想說來寫個看自己智邦免費信箱有多少 unread mail 用的，雖然從 Gmail 開始後就都沒在用了。 先前我都用他自動轉信到 Gmail XD 不過我的智邦免費信箱可是 ms1 的帳號阿! 好像很 cool, 所以還是要多少用一下好了。

其實我覺得, 現在除了 web service 之外, 有個 Desktop application 來結合你的 web service , 這樣對使用者來說會比較 high, 例如: Myshare widget! 這個應該不錯 XD

(謎之聲: 把這個 notifier 改成 URL Notifier widget, 然後到 Myshare 推文, 這樣 blog 就又有流量了 XD)

下面是背面的設定畫面, 設定帳號、密碼跟伺服器, 這是我的 ms1.url.com.tw 的帳號阿! cool 吧! XD

這個 widget 裡面的 POP3 功能是用 perl 掛上 Net::POP3 來作的, 似乎 POP3 沒有辦法只讀 unread mail的數量, 所以這個 widget 顯示的是信箱內 Inbox 裡的所有信件數量, 所以像我不把信砍掉, 或是用 Mail client 收下來的話, 這個 widget 就不太適合。 比較適合平時用 mail client 收信的人。 另外也不適合用 GMail 的人, 我沒有寫 SSL 的東西阿 XD 而且就已經有 Google Notifier 了, 瘋了才用這個 XD

PS. 其實把這個 widget 裡的 POP3 換成 IMAP 應該就可以讀到 unread mail 的數量了, but who support?

anyway, 再來想看看有啥 widget 可以做! 這簡直就跟想做 podcast 又不知道做啥一樣難阿! Orz 最後, 還是得來個 notifier widget 下載, 有興趣的玩玩, maybe bug report? 呵。
