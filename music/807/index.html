Logic Rewire Again!

Apr 2nd, 2007

自從買了 Macbook 之後, 我就不能在 logic express 裡 rewire reason 了, 原本很興高采烈的想試一下新機器的威力(因為 iBook 實在是不夠力了), 但是裝了 Logic express 7.2 跟 reason 3 之後, 居然沒辦法 rewire!! 後來到 Logic 想新增一個 rewire track, 才發現原來 rewire 的 option 是 disable 的狀態, 到網路上找一下, 才知道目前 intel 版本的 logic 還沒有支援 rewire … Orz 後來想說算了, 反正現在 Reason 也是 PPC 版本, 跑起來還是有點慢, 以後再來玩好了。

不過前幾天錄音要新增 audio track 時突然發現 rewire 的 option 居然可以選了!! 才想到先前有軟體更新有更新到 Logic, 發現後馬上更新我的 Reason 到 3.05 (Intel), 挖靠, 雖然還沒有真的去試 rewire 這個東西, 但是, Reason 好快阿!! 想起原本在 iBook 的時候跑 Reason 連 Demo song 都沒辦法撥, 現在居然耗不到 10% 的 CPU Loading, 頓時感受到了 Macbook 的威力阿! 下次來玩玩看 rewire 快不快好了。
