NuSOAP for PHP5

Feb 19th, 2008

由於在 PHP5 已經有 SOAP extension 了, 如果把 SOAP extension 也編進 PHP5 的話, NuSOAP 就會跟 SOAP extension 的 class name 衝到, 所以有了 nusoap-for-php5, 放在 Google code。

不過把 class 名稱改成 soapclientnusoap 會不會有點太長? 我覺得改成 nu_soapclient 會比較好吧? 還是拿 原本的 NuSOAP 自己來改好了 XD

== UPDATE == 剛看了一下 nusoap 的 changelog, 原來因為 NuSOAP 會跟 PHP5 的 SOAP extension 衝到, 所以 class 名稱已經改了勒 o_O, 下面這一段可以看到, 已經從 soapclient 改成 nusoapclient 。

2007-03-19, version 0.7.2 - soapclient: declare as nusoapclient, then also subclass soapclient if SOAP extension not loaded - soapclientmime: declare as nusoapclientmime, then also subclass soapclientmime if SOAP extension not loaded

再來看到這一段,

2007-04-09, version 0.7.2 … - nusoapclient: rename class to nusoap_client …

現在 class name 是改成 nusoap_client 勒, 所以… 記得以後網路上找文章要注意日期阿 XD 像 NuSOAP 只有 code 的話…就只能靠自己多挖依下程式碼了…不然也要看過 readme, changelog 先。

生日快樂!

Feb 9th, 2008

其實對生日這事沒啥太大感覺, 也習慣低調的過, 然而每次生日幾乎都是接在農曆年這時候, 所以同學也都在放寒假每個都回到自己溫暖的家裡, 享受過年的氣氛。 二專跟二技的同學們也沒有人在宜蘭, 也自然沒有人幫我慶生, 不過這樣也好, 我喜歡低調 XD

基本上, 放寒假的時候, 我也沒辦法跟誰說我生日, 沒幾個人知道也是正常的, 不過每年一到生日的時候, 我還是會收到電話、簡訊, 或 MSN 來的生日祝賀, 感謝這幾位朋友記得我的生日 :D
