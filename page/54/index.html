多好笑小海來電.mp3

Jun 26th, 2007 | Comments

話說, 事情就發生在昨天… 就在昨天, 我把 Happy! GO! 錄好之後, 想要馬上傳給志豪聽, 不過志豪居然沒在線上, 我就只好撥電話過去給志豪, 看他在幹嘛, 因為我心裡想說, 待會志豪聽了一定會哭 XD

打電話了過去, 志豪說他跟信宏在彈 bass 跟吉他, 另外還有大貓, anyway, 我就叫志豪快點上線, 想快點給他聽 Happy! GO! 這首歌, 後來志豪就叫我聽信宏的新空心電吉他, 我話都還沒說… 他們就這樣自顧自的彈起來了……..我想一想不對…. 是我打電話過去ㄝ…..我趕緊就說:「 X! 電話是不用錢是不是」….. 最後叫志豪快點上線, 我就掛了電話。

沒想到, 當時志豪居然開著麥克風在錄音, 在我掛完電話沒多久後, 志豪就傳來一個 mp3, 名稱叫做”多好笑小海來電.mp3” 媽的! 我剛才的對話都被錄下來了阿!!!!

我講話很粗魯嗎? 最後我跟志豪討論的結果是, 這才是真性情阿!! XD Anyway 可以聽聽信宏彈的一小段 blues ~

不過不知道誰在最後補一句 “好好笑” = =

(PS. 內有真性情談話 XD 不喜勿聽)

噗~主機 CPU 用量過高

Jun 22nd, 2007 | Comments

噗~ 真的是不知道該高興還是…., 這張圖是在說, 我的主機 CPU 用量過高, 所以我的網站暫時被鎖住了, 其實… 最近這幾天還滿常看到的， 尤其以昨天最為嚴重 XD

我是向 bluehost 租用虛擬主機的, 雖然不像以往看到一些租用 dreamhost 的朋友們的悲慘狀況, 而是暫時的被 block 住, 所以昨天就把主機上一些沒用的服務給關掉, 哈, 其實只有我的 blog 跟 tzangms.com 旗下的零星 blog 有在用之外, 先前架設的論壇也沒人用, 相簿我也是用 Flickr Pro XD 另外, 昨天也把目前 blog 的一些外掛關掉, 把佈景主題中一些不必要的 code 給拿掉, 看來狀況是有好一些。

問題是, 我的 blog 也沒有很多人看阿! 平均一天也才 200 左右的點閱次數, 倒是頻寬流量比較大一點點就是了, 因為放 Podcast, 放自己的音樂, 一個月大概會用掉 10 到 12G 的流量, 所以說, 在 page view 沒那麼多的情況下, 應該是 Wordpress 或是外掛的問題, 看來也許該換個簡單一些的佈景主題吧!?
