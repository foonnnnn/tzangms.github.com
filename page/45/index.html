[Mac] 備份光碟為 Iso 檔

Dec 15th, 2007

今天在裝 Garageband Jam pack 4 的時候, 安裝到一半, 安裝程式就跟我說光碟有損壞, 所以沒辦法直接從光碟裝, 拿出光碟一看, 原來是有刮傷之類的痕跡, 當時想到一個方法搞不好可以, 就是把光碟轉成 iso 再 mount 起來裝, 搞不好可以克服讀取不到的問題, 所以 google 一下, 找到了這一篇 「Create ISO CD/DVD image (.iso) with Mac OS X Tiger (10.4)」

ㄝ~ 我好像只看過 dd 這個指令 XD anyway, 反正都是用 command line 跑內建的指令, 真方便, 本來我還上 google 去找軟體, 好吧! 步驟如下:

  1. 首先把光碟放進 Mac 然後在 terminal 打入下列指令 (看光碟機是哪一個 device, 我的 macbook 是 dev/disk1 )
drutil status
  1. 再來 unmount 光碟
diskutil unmountDisk /dev/disk1
  1. 開始讀光碟轉到 iso
dd if=/dev/disk1 of=file.iso bs=2048

基本上到這裡就可以離開 terminal 的操作了, 用 Finder 去開轉出來的 iso 檔, 可以 mount 起來, 就 ok 啦! …. 可是, 我還是打不開 Orz

嗚~ 我的 Garageband Jam pack 4….

Simplepie in CodeIgniter

Dec 8th, 2007

simplepie, 一個頗好用的 php rss parser library, 因為習慣用 CodeIgniter, 就用 CodeIgniter 的方式來寫一下。

第一個步驟就是到 simplepie 的網站下載套件回來、解開, 然後放到 /system/application/library 裡, 把 simplepie.inc 重新命名成 simplepie.php, 接著就可以像下面這樣子用 (當然, 這是寫在 controller 裡)

$this->load->library('simplepie');
$this->simplepie->set_feed_url('http://tzangms.com/feed');
$this->simplepie->enable_cache(false);
$this->simplepie->init();
echo $this->simplepie->get_title();
