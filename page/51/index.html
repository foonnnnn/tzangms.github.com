IEs4Linux

Sep 27th, 2006

今天Kej大大跟我說了IEs4Linux 這個東西，馬上就裝上來玩一下! 雖然以前就有在用wine，也用wine跑過IE(話說在以前刻苦的時代寫了一篇用wine跑starcraft的文章到linuxfab，只是現在linuxfab沒了，文章被對岸的網站拿去用 XD)，不過這個ies4linux把整個IE套件跟安裝整合的很好，只要執行他的script，然後回答一下yes、no在選個語系，他就會自動幫你抓ie下來裝，可以裝上ie6、ie5.5、ie5還包括了flashplayer9，更屌的是，我切到GNOME看了一下，桌面上還會幫你建好圖示，可以直接執行，整合度超高! 不過如果有人像我一樣使用Windowmaker的話，就直接執行~/bin/ie6就可以啦!

在ubuntu linux 6.06 上安裝的話，只需要先裝上wine跟cabextract就可以直接執行ie4linux的script來進行安裝了，至於速度，一開始開啟ie的時候是有點久，不過一但loading完畢速度還滿不賴的!!那為啥都在linux還要用ie? 沒辦法阿~ 總還是有ie only的網站，為了不重開機，或者根本沒有windows，只好在linux上跑ie囉!

iTzangms Remote 0.1 Release

Sep 27th, 2006

這幾天晚上利用Applescript + PHP寫出來的一支程式，讓Mac使用者可以透過網頁來控制iTunes來播放歌曲。

先前有兩台電腦的時候，一台是PC，一台是iBook，不過平常mp3都放在iBook上，用iTunes來整理、播放，喇叭也都接在iBook 上，又因為出門在外也只能帶iBook出去，所以mp3多放一份到PC上面就覺得多餘，也難整理，所以就都只放在iBook上面。

只是當我在用PC的時候，不可能跑到iBook上面撥歌、換歌，所以當時找了一個iTunes remote，可以直接在PC上面連到iBook用網頁來控制iTunes歌曲播放，這樣一來我就不用在跑來跑去了(雖然在同一個房間裡XD)，只是當初找到的這個軟體要花錢買，但是我又沒錢，所以試用期過就…

一直到上禮拜五，我無聊開始去玩applescript，就上ADC(Apple Developer Connection)抓了一份Applescript Language Guide 的PDF，長達405頁，一開始還很認真一頁一頁看，看不到100頁就跳著看 XD 太累了，呵呵。anyway，後來就想起了先前用網頁來控制iTunes播放歌曲的念頭，乾脆自己寫一個好了!!所以，前三天晚上都在搞懂 Applescript，終於昨天晚上把東西弄出來了，利用applescript讀取的播放列表跟歌曲清單，來輸出到php上，在透過php上的清單透過applescript來控制iTunes，雖然目前很簡單，不過算是夠我用了，其他功能在慢慢寫 XD
