期待錄製中!

Dec 24th, 2005

今天開始在錄期待，之前小廖一次用Guitar pro編了I want to be free跟期待的鼓給我，I want to be free有把鼓給匯入，然後輸出成品，只是 I want to be free的Vocal部份可能需要重錄，因為現在有更好的錄音方式。今天就直接把小廖編好的鼓拿來開始錄音，目前只剩Vocal跟吉他solo的部份還沒有錄，吉他Solo是一個很麻煩的東西，第二段solo簡直就是靈感的考驗，通常一首歌就中間一個solo，所以靈感已經在第一次solo都用光了，碰到第二次就死了XD

這次想說要錄個弦樂的東西進去，其實之前就有發現到一個問題，縱使你有弦樂的音色，但是要用midi玩出小提琴的感覺，並不是怎麼彈都可以像小提琴，撇開樂器上的限制不說 (例如用KB彈的時候，要考慮到實際上小提琴跨弦的問題、甚至把位…等等) 要彈出小提琴感覺，還是要知道人家小提琴怎麼拉，拉什麼音或樂句。所以說並不是光靠器材就可以解決的問題。最近一直在聽有配弦樂的東西 :D

PS : 蘇打綠的歌裡面都滿滿的弦樂
