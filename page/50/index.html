語音聊天小撇步

Sep 8th, 2007 | Comments

如何用筆記型電腦透過 skype 聊天不戴耳機, 又不怕有回授? 看到 這篇文章 寫了個小技巧, 應該滿有用的。

其實他最主要說得就是, 如果你的麥克風在右邊, 那就把喇叭的聲道調到左邊, 相反的, 如果麥克風在左邊, 就把喇叭的聲道調到右邊, 這樣一來喇叭跟麥克風就有些距離, 可以減少喇叭出來的聲音直接回授 (feedback) 到麥克風的情形!

不知道怎麼調喇叭聲道的話, 可以看看原文, 有圖解說明, Windows 跟 Mac 都有喔!

哈, 之前都沒想過這個, 所以每當聊天的時候, 如果對方說我這邊都有回音, 聽起來很難過, 我就會接上耳機, 可是就感覺有那麼點受限制, 不是很舒服, 不過看來下次語音聊天, 可以試試看這個撇步 XD

只是我的 Macbook 的麥克風在中間 … 好吧, 有點偏左 XD

關於 SVN 的備份

Aug 30th, 2007 | Comments

這裡是針對單一個 SVN repository 作備份, 可以匯出成單一個檔案, 不過如果你 SVN 是用 FSFS 當作儲存後端的話, 應該是可以目錄整個 tar 起來備份的。用 Berkeley DB 的話, 應該就沒辦法這樣做。(阿災, 沒用過 Berkeley DB, collabNet 也建議用 FSFS)

備份指令如下:

svnadmin dump foobar > foobar.dump

還原指令如下:

svnadmin load foobar < fooar.dump

例如我的 svn repository 是放在 /var/svn, 然後有個 project 叫做 stickeraction 那麼我要 dump 就是下這個指令

svnadmin dump /var/svn/stickeraction > stickeraction.svndump

那要還原的話就是

svnadmin load /var/svn/stickeraction < stickeraction.svndump
