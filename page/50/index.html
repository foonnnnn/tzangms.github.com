CodeIgniter Mssql Driver 補完 XD

Jul 5th, 2007

最近用 CodeIgniter 搭配 MS SQL Server 在寫程式, 把原來是針對 MySQL 的程式拿來改成用 SQL Server 當資料庫, 不過卻碰上滿多問題的, 特別是 SQL Statement 的部份, 有幾個看起來不會有問題的 statement 全都因為資料庫換成了 SQL Server 而產生了錯誤。

不過在 CodeIgniter 上面用 ActiveRecord 其實已經去除掉一些可能會發生 Statement 不相容的問題了, 不過後來才發現, 原來 CodeIgniter 雖然支援許多資料庫, 但是其實對於 MS SQL Server 的支援並不完整, 當然是碰到了才知道, 原來 insert_id 根本沒有實作, 後來到 CodeIgniter 的論壇找了一下, 有人已經提出來了這個問題, 也有人提供實作好了的 insert_id patch。可以看到這一篇。

格強有耳

Jul 4th, 2007

話說昨天下班我就衝去上 DearJohn 的廣播節目- 格強有耳, 呼~ 真的是非常的緊張, 果然跟自己一個人在家裡錄 Podcast 的感覺差很多, John 的口調真好, 真該跟 John 多學習一下。

在節目中播放了自然而然跟眉頭這兩首歌曲, 其實在節目中應該是因為太緊張又想搞 kuso 的關係, 有些地方有點胡言亂語, 結果在現場唱歌又因為太緊張, 所以聲音放不開, 還會發抖勒…Orz 我想我會寫一首新歌, 叫做”只因當時太緊張” XD

另外在節目中我還唱了”背叛”這首歌, 說真的, John 這篇介紹 ”來聽比楊宗緯還好聽的「背叛」吧！” 的標題真的是讓我冒冷汗阿! XD 呵, 在自彈自唱”背叛”這首歌時, 因為先前根本沒背好和弦, 所以彈錯啦!!! XD 所以我怕到時候我會被楊宗緯的歌迷圍剿, 呵呵(苦笑) 不過這一次到中時錄音的經驗還真的是滿好玩的:p 感謝 john 給我這個機會呢!

格強有耳這個節目是在每週四、週五的中午 1:00 播出, 到時候大家記得去線上收聽 XD 如果沒辦法聽的, 沒關係, 之後 John 也會把節目放上他的 Podcast, 所以大家現在就可以馬上到 John 的網站去訂閱 Podcast XD

最後, 我怕因為 John 的介紹來到這裡的朋友們, 找不到我的歌可以聽, 我又把右邊的播放器放出來了, 裡面都是我寫的歌, 有興趣的去點來聽聽吧! XD
