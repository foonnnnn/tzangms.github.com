The PHP Benchmark

The PHP Benchmark 。 有寫 PHP 的話, 這個站一定要看一下的阿! 其中有些 benchmark 好像推翻了先前所讀過的東西。

像是以前讀過說, while 比 for 快, for 又比 foreach 快 (雖然我還是一直用 foreach, 因為 code 比較乾淨 XD), 不過這個 PHP Benchmark 在測試 loop 的結論提到, 其實 foreach 是比較快。

for ($i = 0; $i < count($foo); $i++) {
    // blah
}

還不錯用的 jQuery Template

Jun 6th, 2008 | Comments

隨著 AJAX 用的越來越多, 覺得 javascript 真的是需要一個 template 的機制。由於平常都用 jQuery, 所以找了一下 jQuery template, 找到這個 jQuery Templates, 裡面只簡單提供了一個 js 檔, 也沒有說明, 不過打開 js 檔來看, 其實說明跟用法都在裡面了, 參考如下, 試了一下還不錯用。

var t = $.template('
Hello ${name}, how are you ${question}?  I am ${me:substr(0,10)}
');

$(selector).append( t , {
   name: 'Stan',
   question: 'feeling',
   me: 'doing quite well myself, thank you very much!'
});
