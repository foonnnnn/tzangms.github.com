[Demo] 我想忘記你 (新版)

Apr 10th, 2008

前幾個禮拜, 我把我想忘記你這首歌, 不管是吉他、Bass、鋼琴…所有的樂器跟 Vocal 都從頭到尾都重新錄了一遍, 而且還升了半音。 錄玩之後, 兩個版本一對照起來, 我聽了連自己也嚇了一大跳! 原來的版本是完全只用  Logic 7 express 內建的外掛來錄製, 像是電吉他跟 Bass 用的是內建的 Guitar Amps 來當效果器, 鼓則是用 GarageBand 的 Drum kits, 鋼琴也是, 而當時的 Vocal 也是用一支 $199 的便宜麥克風和不到兩千塊的音效卡, 然後用以前舊的 iBook G4 800 來完成整首歌的錄製。

當時, 或是說是在幾個禮拜前, 我認為原來的版本就不錯聽了, 不過現在聽新錄製好的歌曲, 一聽就可以聽出來, 這兩個版本之間, 錄音品質的差別有多大! 

這次在重新唱跟重新錄製這首歌的過程中, 由於錄音品質差太多, 所以在這當中最難抓的就是原本歌曲所營造出來的感覺, 畢竟也聽習慣了原先錄製的版本。所以不管是在唱, 或是後製的過程, 都吃足了苦頭。 要在錄製那麼近、清楚的狀況下, 再去加入原來版本所營造出來的空間感, 真的是兩難。不過在經過了非常多天的嘗試跟後製, 總算弄出了我想要的感覺, 有了現在這個還算不錯的版本 :p

不過其實現在聽到的這首歌, 還沒算完全後製完成, 還是稍有些小瑕疵, 而且整首歌也還沒達到讓我滿意的程度, 不過我就是想放上來給喜歡聽我的歌的人聽聽 :p 老話一句, 這首歌也是很適合睡覺聽 XD

歌詞跟原來的版本可以到原來的版本聽聽、看看。

PHP 語法錯誤檢查

Apr 3rd, 2008

之前在 Tsung 那邊看到 大量檢查 PHP 程式是否有 Syntax Error 這一篇文章, 後來每次想到要檢查的時候, 都因為這一段太長記不下來 XD 所以我每次就又都從 Google 找這篇文章, 今天想說把它寫下來, 寫成一個 shell script 好了 XD

我把檔名叫 phplint #!/bin/bash find . -name *.php ! -exec php -l {} ;

在 chmod +x phplint 之後, 在想要檢查的目錄打 phplint 就可以了。

基本上我是在自己的 home 有開一個 bin 的目錄, 然後再 .bashrc 裡加上 export PATH=$PATH":~/bin"

之前的 twit shell script 也是這麼作 :p

[PHP] XML Schema 驗證

Apr 1st, 2008

記得之前再翻 PHP Manual 時, 看到 DOM 裡有 schemaValidate 這個東西, 也因為工作需要, 所以來試一下, 還真方便好用。

$xml = file_get_contents('http://tzangms.com/foo.xml'); $dom = new DOMDocument; $dom->loadXML($xml);

$is_validate = $dom->schemaValidate('bar.xsd');

或是用 load 讀 xml 檔名

$dom = new DOMDocument; $dom->load('foo.xml');

$is_validate = $dom->schemaValidate('bar.xsd');

另外 xsd 的部分, 也有 schemaValidateSource 可用 $dom->schemaValidateSource($xsd);
