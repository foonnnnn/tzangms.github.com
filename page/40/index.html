好忙! 來點靈感吧!

Aug 29th, 2007

最近真的是太忙了, 每天回家就是在 K 這本「Pragmatic Version Control Using Subversion, 2nd Edition」, 本來是想用來管理自己的程式碼, 才在公司架 SVN 來用。

而最近在帶公司新進來的 Programmer, ken, 要檢視他的 code, 幫忙更新, 雖然幸好有 SVN, 幫上我很多忙, 可以很方便的就知道 ken 改了哪些程式碼, 看完 ok, 就把程式碼 commit 到 SVN, 接著更新到線上的機器, 但是慢慢發現不強迫 ken 也用 SVN 不行了, 當他修了 2~3 個 bug 都沒有 commit changes 的時候, 我就沒辦法知道哪些 code 是修同一支 bug, 更何況我有時候也要修同一個 project 的 bug, 所以在同一個 working copy 下改 code 就會很麻煩。所以教他用 SVN, 建立一些規則, 這樣之後程式碼的管理及更新就方便多了, 當然最好是可以弄到 auto deployment。

Anyway, 以前只有一個 programmer 在 maintain 一個 project 的時候, 就不會有什麼問題, 但是公司的 programmer 越來越多了! 最近公司多了一位顧問, 然後 10 月又會來一位新的 Programmer, 所以我就意識到, 媽的! 先不 K Ruby 了, 開始每天下班回家猛 K SVN!! 甚至這幾天我連上班都在 K。 目前除了手上的 Project 要弄, 每個禮拜都要開會, 然後要帶新人, 檢查他的 Code、更新, 然而這禮拜又加上值班, 還有一些零星的 request, 其實真的是有點喘不過氣。 (好吧! 帶新人只是個過度期….)

阿!! 我也不算舊人阿!! Orz

所以最近 twitter 少 mur 了許多, blog 也沒時間寫, msn 每天掛忙碌或離線, 昨天才抽空彈一下吉他, 錄下點靈感, 呵, 不過聽起來應該跟亂唱沒啥兩樣, 其實還有錄另一首, 不過這首比較可以放上來 :p 只是….副歌唱一半….聽起來應該會很難過 XD 哈哈哈, 沒辦法…好忙阿 Orz

搬家成功!!

Aug 21st, 2007

由於這個週末整個掰咖, 我只能在家裡打電腦、上網, 不是打開 bloglines 看看, 就是 twitter, 當然自己的 blog 也要看看, 但是自己的 blog 越看就越生氣, 整個慢到不行, 想想剛開始用 bluehost 的時候還滿快的, 當初也很滿意 bluehost 的速度, 怎麼感覺越來越慢了, 我的天阿!!

查一下自己的 bluehost 還剩兩百多天可以用, 但是 blog 載入到一半就掛了的這種速度, 我實在是沒辦法繼續用下去, 所以心裡就想說:「媽的! 一定要搬家!」

所以一開始我就先把 h2oband.net 這個 domain 先 transfer 到 GoDaddy.com, 先來試看看, 反正之後要搬到那裡, 直接指過去就好了, 另一個原因是因為… bluehost 的 domain 管理真的很陽春, 只能指 name server, 沒辦法設 CNAME, MX 之類的東西 (也許是我沒發現 XD), 所以我就先把 h2oband.net 這個目前沒在用的 domain 先 transfer 到 GoDaddy 試看看, 沒想到~ 居然發現 GoDaddy 有在賣 hosting!!

在處理 h2oband.net 的 domain transfer 的時候, 感覺 GoDaddy 的速度感覺還滿快的, 相對的, bluehost 就一整個慢到爆! 就想說先在 GoDaddy 買個一個月玩玩看好了, 結果~ 馬上就花了 $6.99 美金, 噗~

Blog 串連罷免宜蘭縣長呂國華

Aug 8th, 2007

真的是每天都有新鮮事, 最近累積了好多文章想寫, 誰知道又來這個東西。 就是說, 今天一早打開 Bloglines 就看到 Mark 的 blog 的一篇文章: 「Blog 串連罷免宜蘭縣長呂國華」 這個標題馬上吸引了我, 嗯, 沒錯, 我是宜蘭人阿! 在看到 Mark 的這篇文章之後, 我就馬上把他收到 Myshare, 希望有更多的宜蘭人可以看到…

我想大家應該都有在我 Blog 的右邊有看到兩張貼紙, 這兩張是在今天中午 Jas 做出來的, 超有行動力!! 其實在早上看到 Mark 的文章之後, 我就馬上去找了一堆相關新聞來看, 然後就想說今天一定要發文來串連這篇活動, 才想到…沒有貼紙阿! 沒有貼紙怎麼可以算得上是串連活動呢!? 所幸 Jas 馬上就在他 Blog 發文, 貼出這兩枚精美的貼紙, 該稱這兩枚貼紙為精美嗎? 對我來說, 這兩枚貼紙充滿著宜蘭人的憤怒阿!! 宜蘭童玩節舉辦了十二個年頭了, 現在突然說要停辦, 真的是很氣人!

「我不知道宜蘭有什麼好玩的。」當朋友問起我宜蘭有哪裡好玩的地方, 我總是這麼回答他們。 但是我還是會擠出幾個東西, 其中一個就是童玩節。如果童玩節沒了, 你要我還能說出宜蘭有哪裡好玩? 好吧! 這是我個人膚淺的玩笑話。童玩節, 我想台灣北部大多數的人一到暑假, 都會想到宜蘭的童玩節, 不是嗎? 好吧! 至少我是, 我身邊的朋友也是, 如今卻為了這麼爛的理由, 這個爛縣長要停辦。

就像 Mark 跟 Jas 所說的, 碰到問題不去解決, 而是決定放棄、停辦, 那麼請你下台!

身為宜蘭人的你, 如果你有 Blog, 也同樣對這件事情感到氣憤, 煩請加入這個串連活動! 在你的 Blog 貼上這兩枚貼紙。如果你不是宜蘭人, 但也關心宜蘭童玩節的話, 你也可以只貼上「搶救童玩節」的貼紙, 「罷免呂國華」的貼紙就交給我們宜蘭人吧!!
