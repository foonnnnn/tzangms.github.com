Beginning CouchDB

Jan 10th, 2010

上上週週末讀完了這本書, 真的是很 Beginning level, 還滿好讀的, 所以一下子就讀完了, 大概 300 多頁。 可能是 CouchDB 設計的真的很方便, 內建了 Futon 這個 web 介面可以用來管理資料庫, 從視覺上的確比較容易了解整個資料的架構。

第四章, 示範了用 curl 來對 CouchDB 操作的 CRUD, 超上手! 感覺真的很方便, 所有的動作都透過 curl 就可以達成。

第五章介紹使用 Futon 這個介面, 整個感到 Amazing, 怎麼這麼方便呀! 其實書中圖滿多的, 所以頁數也佔掉很多, 但就還滿容易看的, 因為一下子就翻了一頁, 感覺很有成就感 XD

第七章是 CouchDB 的 Views, 第八章則是 Map/Reduce, 這兩章就是 CouchDB 的重點了, 好像 Map/Reduce 也沒那麼難懂 (狀態顯示為第一次讀 Map/Reduce ) XD 或者這真的是 beginnng level o_O

最後還有提到 CouchApp 以及 Couchdbkit ( python ) 跟 CouchRest ( ruby ), CouchApp 還滿值得看一下的耶, 而 couchdbkit 就提供了一個 ORM like 的 Document 定義方式, 也有提供 django 用的 module, 用法就跟 django 的 ORM 類似, 甚至還有提供類似 ModelForm 的 form 喔!

不過 couchdb 不像 mongodb 有 collection, database 下就直接是 document, 而書中也沒有提到一些使用上的 scenario, 目前還不知道可以用在哪比較好。

其實對 mongodb 比較有興趣 :p

將我的第一部機車報廢了

Jan 7th, 2010

自從換了現在這份工作, 每天就都是坐捷運上下班, 因為我也不喜歡台北的交通, 可能習慣宜蘭暢通無阻的道路, 所以感覺在台北騎機車還滿恐怖的, 現在坐捷運上下班, 所以也很少有騎機車的機會, 而這兩三個月我的機車就一直擺在路邊的停車格, 甚至我也忘記他的存在 XD

今天終於完成報廢的最後的一步驟了, 其中的心路歷程就不多說了, 18歲到現在, 十年的機車, 記錄一下。

還記得當初高三的時候, 剛滿 18 就買了這台機車 - 三冠王, 然後就開始騎機車上下學, 幸好都沒被教官抓到。 記得當初還有在純精路上把另外一台機車整個連人帶車撞飛出去, 超爽的, 我完全沒事, 而且也不是我的錯, 那個人爬起來看了我一眼就又騎著機車走了 XD

畢業之後考上在苗栗的二專, 就把機車從羅東火車托運到竹南火車站, 因為學校在山上, 所以然後就靠這台機車上下山, 沒事就往新竹市區跑, 跑北埔跟同學出去玩之類的。 寒暑假就又運回家, 然後常往宜蘭跑, 因為當時女朋友在宜蘭, 當時騎上 40 分鐘也不覺得累 XD

然後二技考上了在新竹湖口的學校, 一樣把機車透過羅東火車站托運, 不過這次是託運到新竹火車站, 記得托運都會碰到車鎖被撬開到 on 的狀態, 然後電池耗到沒電的情形, 火車托運真是爛透。

在湖口之後也是常騎著這台機車往新竹市跑, 或者是往桃園、中壢或是新屋跑 :p 一直到畢業時, 想說學生的生涯告終, 我也不想託運機車, 所以我就騎著機車, 一路從湖口騎回南方澳, 真是累死人了, 我似乎花了 9 個小時, 然後回到家裡的時候, 騙家人說我之前就把機車托運到羅東, 我是做火車回到羅東, 直接騎機車回來的 XD 後來也載著當時的女朋友, 一路從南方澳騎到福隆去看第三屆海洋音樂祭, 路上真是塞爆 ~

其實在從湖口回南方澳的路上, 經過他媽的台北, 我路上經過了好幾次同一座橋, 然後從同一個角度看到台北 101 好幾次, 覺得這條路好熟悉之類的。 原本我以為照著路標走就可以回到宜蘭, 誰知道台北的路標真的他媽的一個亂, 最後 blah blah 的, 似乎快接近北宜公路時, 我問了怎麼北宜公路怎麼走, 最後找了間麥當勞吃個東西休息, 然後繼續上路。

誰知道他媽的, 在北宜公路中我的機車快沒油了, 然後機油又亮紅燈, 啃, 已經騎了好幾個小時了耶~ 我真怕他會罷工, 不過當時我的手已經快握不住機車握把了, 然後一路上的荒涼的景色, 擔心著我的機車快沒油, 路上完全沒有加油站, 機油又亮紅燈, 擔心機車會爆炸, 手又快握不住了, 不過也只能硬著頭皮騎下去。

幸好終於騎到了似乎像是坪林的地方, 終於有人煙了~ 所以趕緊找間機車行換機油, 然後加油, 還有在 7-11 買個東西吃, 順便休息一下, 停下車子後, 感覺整個身體似乎不是我的了 XD 最後繼續往礁溪、宜蘭、羅東一直騎回家, 花了我 9 個多小時阿!!! 最後手抖個不停, 身體都還是一直以為有風在吹著。

然後我去當兵, 這台機車就被我弟送到台北供他讀書可以騎, 一直到我退伍在台北找了工作, 我弟就把機車從中永和騎來內湖還我, 然後他回宜蘭, 只是沒想到機車已經傷痕累累, 當初交出去不是這樣的阿!! 居然金屬製的煞車居然還有整個削掉的痕跡, 是怎麼騎的阿!!!

之後這台機車就讓我台北到處跑, 一直到今天, 他終於功成身退了, 再見了! 我的三冠王 LEO ( 車牌號碼 )

Libcloud - Cloud Service Manage Api

Jan 7th, 2010

剛才在消化 RSS 的時候看到的, 沒想到有這個東西。 libcloud 對一些 cloud service 管理的 api 提供了一個統一的介面 ( python library ), 支援一些較知名的廠商像是 ec2, slicehost, linode, rackspace, gogrid 等。

可以做以下動作:

  • list
  • reboot
  • create
  • destroy
  • images
  • sizes

過一陣子應該有時間可以來玩一下, 試試看 linode, slicehost 或 ec2 之類的, 感覺滿好玩的 XD
