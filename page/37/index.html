[網摘] May 16th 2008

May 16th, 2008 | Comments

The Web Developer’s SEO Cheat Sheet 這個 Cheat Sheet 還有 HTML 的語義說明, 我覺得這個 Cheat Sheet 真的是做的很棒! 下載 pdf 印出來放在桌上吧! XD

What makes a good application framework 看看這篇的標題, 這就是為何我選 CodeIgniter 這個 php framework 阿!! framework 就是要好上手, 文件好懂, 不會綁手綁腳阿!! 如果說要用一個 framework 還要花上好幾個月來學, 那麼就不快速了。

Patterns for PHP 這真的是好站阿!! 先前就想找 Design Patterns 的書來看, 像是 Head First Design Patterns, 只是市面上的書大多都是以 Java 來做範例。 如果不知道 Design Patterns 是用來幹麻的, 可以看看 Tsung 寫的這一篇 好站: Design Pattern for PHP 的 範例實作與解說

UI-patterns.com 除了程式的 Design Patterns 之外, UI 也有 patterns :p

Regular expression tool 這真的是一個很讚的線上 RE 工具阿!! 可以選用 PHP PCRE, PHP POSIX, Javascript , Perl 跟 Python 等 RE 模式, 還可以選用 Modifier, 真是不錯!

Automatic config[base_url] - CodeIgniter Forum 通常在開發時, 都會有三個環境, development, stage 跟 production, 但是當使用 CodeIgniter 在開發時, config 中 base_url 就寫一個, 因為我使用 capistrano 來 deploy php 程式到 stage 跟 production, 所以這不造成困擾, 因為 stage 跟 production 有另外的設定檔, 但是有多個 development 的環境的話, 就可以看看這篇的作法, 其實應該不用看就很多人都自己改成這樣了吧?
