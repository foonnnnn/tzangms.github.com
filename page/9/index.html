Vimium

Apr 16th, 2010 | Comments

vimium 真的是超讚的!! 早上看到 jas 的新文章 後, 今天到公司就從 safari 換到 chrome, 用 vim 的指令來瀏覽網頁, 對於平常都用 vim 來工作的我來說, 一整個大開心! 之前就一直想要擺脫滑鼠, 因為用滑鼠讓右手跟右肩很不舒服, 而且現在手都不用離開鍵盤, 真的是很讚! 用了一會, 現在大部分的動作都不用滑鼠, 而且本來就很習慣 vim, 現在一整個順, 哈哈哈哈!!

不過現在因為 web 2.0 有些動作會是滑鼠 hover 才會出現, 或者部分 ajax 動作不是透過 form button 或 anchor, 所以有些動作就還是得透過滑鼠。 像是 google reader, 就得先透過滑鼠來進行點擊後才能用 hjkl 來切換文章。

不過碰到可惡的 flash, 點了之後會把 focus 搶走, 用 tab 也切不出來 … 所以 vimium 也會失效 …

Anyway, 現在還算滿 high 的 (笑)

Twitter Anywhere 試玩

Apr 15th, 2010 | Comments

今天玩了一下 twitter anywhere, 只要照著文件 加個幾行 javascript, 很容易就可以弄出來了。

目前有下面幾個功能

  1. Linkify
  2. Hovercards
  3. Follow Buttons
  4. Tweetbox
  5. Connect

Linkify 跟 Hovercards 的功能差不多, Linkify 就是自動幫你把 @ 加 twitter 帳號的字串, 自動加上連結。 而 Hovercards 基本上就是 Linkify 不過加上了當 mouseover 的時候, 會 popup 一個 twitter 的 user info 的功能。

例如:

@tzangms 或 @stickeraction 甚至是 @myaudiocast XD

至於 Follow Buttons, 可以看到我部落格左邊的 button, 可以直接讓人家按下去 follow。 至於 Tweetbox 我就沒試了, 倒是 Connect, 幹, 文件根本就寫錯嘛!!! 目前是可以 connect, 可是 connect 之後卻取不到 user 的資訊, 就等文件更新了 … anyway, 大概是這樣。

不過大家還是可以先按下面的按鈕, 先跟我的 blog connect 一下 XD
