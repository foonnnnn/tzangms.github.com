原來我學過這些程式語言

Jan 24th, 2009

從高中開始接觸電腦, 一開始就是上網, 然後開始接觸 HTML, 因為聊天室可以用 font, 覺得很厲害, 然後開始學怎麼做網頁, 然後學 Photo Impact 3 然後又換到 Photoshop 4, 然後接著用 Dreamweaver 來作網頁, 就這樣開始對網頁製作有興趣, 畢竟在用電腦之前, 我都是在畫畫。

接著高二開始碰 Linux, 然後學校的課程開始教 QB 然後 VB, 後來高三的課程甚至要用 VB 來處理圖片的明暗度平衡, 現在想起來真是厲害, 只是寫不出正確的處理方法 XD 而高中除了 VB 之外還學組合語言, 單晶片 PIC 阿什麼的。

然後上了二專, 仍然是組合語言, 8051, 然後還有 PC AT/XT 界面設計, 所以學 C 語言, 然後還有 VHDL, 一直到了專題用了 C 語言寫了一個射擊遊戲過關 XD

到了二技, 程式設計的課程變少了, 只有教 C 而且居然教最基本的, 但是我一點都不在意, 因為那時都在玩樂團, 然後到了最後一年, 有一門 3D 圖學的課程, 我用 java + swing 來交作業, 現在想想還真的是不可思議, 我居然會寫 java。 然後專題我負責的某個部份是用 Flash Communication Server, 所以我就開始摸 Action Script, 寫 Client / Server 端, 記得當時只有一本原文書在講 FCS, 只是後來也沒人用就是了。

以上其實是因為在二專跟二技我幾乎所有的時間都在玩樂團… 所以… 對以上這些東西沒有太大印象, 所以想起來, 自己才覺得訝異, 寫出來紀念一下。

Master One, Then Learn Another

Jan 24th, 2009

Master one, then learn another

這句話忘記從哪裡看來的, 基本上是說, 精通一個程式語言以後, 在去看看其他的。

其實這一篇完全無重點, 碎念類。

我主要使用的語言就是 PHP, 但是完全稱不上精通, 畢竟也才兩年半的時間, 從我的第一份工作, 才開始學 PHP。 記得剛進公司每天都是開著 php.net 查 manual, 當時連 substr 都不清楚, 而 HTML 的 FORM POST 也不太清楚怎麼運作。 而 SQL 語法也是一直上網查, 除了 SELECT 之外都不會。

然後工作了一段時間, 大約半年才開始學 javascript, 然後對 javascript 不清不楚的狀況下就開始用 jQuery, 一直到了大概半年多前開始讀了 DOM scripting 這本書後, 才對 javascript 稍微了解一些, 才懂得 document.getElementById 這類的東西。

而我的 PHP 也是很弱, 一直到了大約一年前開始用 CodeIgniter 之後, 才慢慢的解決我心裡對 PHP 的焦慮, 因為 SQL Injection, XSS 等, 都交給了 CodeIgniter, 然而之前學了 ruby, 最近學了 python 跟 django, 只是似乎對 PHP 的部份還是很弱, 似乎還不到去學其他語言的地步。 也許該來好好學一下 PHP 吧。

又忘記哪裡看到的一句話說:

你可以用 library 跟 framework 來減輕你對程式語言能力不足及焦慮, 但當你使用這些工具之後, 慢慢不在感到焦慮時, 你該去研究你用的 library、 framework 來補足你還不清楚的地方。

Anyway, 先前在學 ruby 跟 python 時, 還滿有啟發的, 看到了很多不同的東西, 使用的程式語言不同, 想法、思維也跟著不同。 只是 PHP 才是我目前唯一能謀生的東西, 所以該回頭來好好研究一下 PHP 吧。

Amazon S3 使用記錄

Jan 23rd, 2009

昨天開始碰 Amazon S3, 因為一直覺得 stickeraction 的貼紙圖檔放在 VPS 上還是不太安全, 所以前天就開始用 Amazon S3, 把貼紙圖檔都丟上去, 順便當 cdn 用。

一開始是先看 Tsung 大大的文章, 然後就開始用 s3cmd 開始玩看看 S3 怎麼用。

首先就是在 gentoo 上裝 s3cmd 啦, 不過試了 x86 跟 amd64 的平台好像都是 masked, 所以就用 ACCEPT_KEYWORDS 裝起來

ACCEPT_KEYWORDS='~x86' emerge s3cmd

然後就是用 s3cmd --configure 這個指令來設定, 填入 Amazon S3 提供的 access key 跟 secret key, 接著就可以開始用 s3cmd 來丟檔案到 s3 上做測試了。

我站在河岸留言舞台上

Jan 10th, 2009

這張照片對我來說還滿有意義的, 感謝叮咚幫我拍這張! 真的! 超感謝!

以前在學生時代玩團時, 就很想到台北表演, 想要到 the Wall、地下社會、海邊的卡夫卡、女巫店跟河岸留言, 這些聽獨立音樂的人都一定知道的地方表演, 終於我也站上河岸留言的舞台, 唱起自己的歌了。

雖然這對大部分在台北的樂團、音樂人來說似乎不是大不了的事。

只是台北跟台北以外的音樂資訊落差真的頗大, 當台北的音樂人都在唱著自己創作而不唱 copy 時, 在台北以外的地方, 卻呈現相反的狀況, 我在二技時唱著自己的創作, 每次得到的回應都是: “怎麼都唱一些沒聽過的歌” 不然就是 “我想聽周杰倫的雙節棍” …. 但是我還是這樣唱了好久, 唱到大家都會跟著我一起唱, 一起唱我的歌。

當時覺得在台北唱歌仍然是遙不可及的, 一直覺得程度差太多, 怎麼可能有辦法到這些地方唱。 不過當時就打定主意, 決定將來一定要到台北找工作, 在台北玩團, 這樣我一定也可以到這些地方唱我自己的歌。 只是等到鼓手去當兵退伍後, 貝斯手也準備要去當兵了, 我的樂團似乎怎麼也搭不起來。

最後誰知道我現在把時間都花在寫程式了。

幹, 下一個人生的目標就是 … 出第二張唱片, 然後把這些地方全都唱過一遍!
