Google AppEngine 試玩

Jun 18th, 2008

這兩天下班後, 都在玩 AppEngine, 不過我不太會 python, 也不懂 django, 只有上禮拜花了一天晚上看完 python tutorial, 就開始亂玩 ~”~ 所以寫了一個 twitter easy unfollow 的 twitter 應用 XD (因為實在太不穩就不放連結上來了 :p )

在寫的過程中一直碰壁, 一方面是 twitter 最近太慢, 另一方面是 appeingine 的 urlfetch api 似乎只有 5 秒的限制, 所以加上 twitter 太龜, 一直掛點, 還有碰到 DeadlineExceedError… 我只不過用了 3 次 urlfetch, 把 fetch 回來的 json 轉成 dict, 然後 append 起來而已, 接著就送 template 了阿!!! Orz

anyway, 在開發的過程當中用了一個使用 memcache 當 session 的 library ( http://code.google.com/p/gmemsess/ ), 還真方便 XD

剛剛 ericsk 大大也丟了一個連結給我 ( http://code.google.com/p/google-app-engine-oil/ ), 原來是大大為 AppEngine 弄的 MVC Framework。 ericsk 大大, 我會用看看的!
