[網摘] March 25th 2008

Mar 25th, 2008

  • Question your work

當你要開發產品的新功能的時候, 曾經想過為何要開發嗎? 這個功能是否真的有用? 帶來的價值是? 更重要的是它是否符合使用者的需求? 也許跳出來這個框框, 先看看你自己覺得好不好用?

  • 10 tips to make your PHP application scream

一些關於 php.ini 設定的東西, 像是 HTTP_POST_VARS 你還會用到嗎? 不用就關掉它吧!

  • The Secrets Of Grunge Design

關於 Grunge style 的設計風格, 有人說這是 web 2.0 之後網頁設計的新趨勢?

  • HTML Purifier

用來過濾 XSS (filter)的 PHP library, 另外在它網頁上也列了出來其他相同功能的產品的比較, 說明的滿詳細的, 也有 Wordpress 的 plugin, 另外也提到有 CodeIgniter 的 library, 但是其實只是說明, 如何在 CI 裡使用 HTML Purifier, 可是我覺得它範例所寫的使用方式, 覺得有點遭, 另外, 雖然這支 library 功能很強, 但是…好胖 XD

  • Really cool CSS diagnostics script

這真的還滿酷的, 只要掛上這個 CSS, 就可以在網頁上看出哪些東西、屬性是需要修正的, 有稍微玩一下, 有空再來研究看看。

  • Simplicity On The Web: Designing To Hide Features

做 UI 設計應該可以看一下, 其中有介紹一些知名網站的範例, 避免一次給使用者太多資訊。
