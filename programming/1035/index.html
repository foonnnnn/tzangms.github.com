Markup Validation Service

Dec 20th, 2007

從 Tsung’s Blog 看到 這一篇文章「好站: HTML/XHTML/CSS 驗證」之後才想到, 作好的 PP 都還沒有拿來測過, 所以昨天晚上就把 PP 丟過去測看看, 沒想到 error 還滿多的, 想說不知道其他網站的狀況如何? 乾脆再拿一些網站的首頁丟進 Markup Validation Service by W3C 測看看。

就來看看一些網站的數據吧: (想到就丟進去測了)

  • tzangms.com
    Passed validation
  • HEMiDEMi
    Failed validation, 72 Errors
  • PP
    Failed validation, 81 Errors
  • funP
    Failed validation, 96 Errors
  • MyShare
    Failed validation, 302 Errors
  • wretch
    Failed validation, 582 Errors

其實之前才因為要做 SEO 才對自己 blog 測過, 結果當然是 Perfect! 一個 error 也沒有! XD 嗯, 除了可以做 SEO 之外, 我不知道 pass 這個 markup validation 有何其他好處? 因為現在這樣看起來, 一些知名的網站也不太管自己的網頁是否 pass 這個 validation

anyway, 另外昨天 jas 也有在 twitter 上提到 XHTML-CSS Validator 這個網站, 同時也可以看 CSS 是否 validate, 把我的 blog 丟進去測, 呵, 看來我的 CSS 不行阿 XD
