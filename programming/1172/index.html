不用 Twhirl 也可以 Retweet ! (GM Script)

May 19th, 2008 | Comments

因為羨慕 twhirl 的 Retweet 功能, 所以寫了一個 GreaseMonkey 的 retweet script。 掛上這個 retweet script 之後, twitter 頁面就會顯示如下(多了一個 R 的按鈕):

感謝無敵小恩恩給截 twitter 截圖, 他正在”拉正”學弟 XD

之後只要看到想要 Retweet 的訊息, 只要按下上圖所標示起來的 R 就可以 Retweet 了。

不過由於今天我第一次裝 GreaseMonkey, 裝完之後就矇著頭開始寫, code 很亂~ 我想 XD anyway, 之後有空再改吧! 目前先可以用就好了 XD

如果不想裝 twhril 但又想用 retweet 的話, 就試試看這個吧! 但是前提是你瀏覽器要用 Firefox, 如果你是用 Firefox 的話, 就先到這裡裝個 GreaseMonkey 吧! 等 GreaseMonkey 裝完再點下面這個連結, 裝完就可以開始用囉!
