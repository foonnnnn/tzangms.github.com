新玩具: Instawall

Feb 20th, 2011

http://instawall.djangy.com

星期六早上收到了先前 instagram 發來的 api developer 啟用信, 就開始玩了起來。目前把 自己的照片 / 朋友的照片 / following / follower / popular / like / comments / 加好友 等功能都做上去了。

所以現在可以直接從網頁上按 like 跟留言了, 不過加好友的部分 api 好像有點問題, api 回應正常, 不過好友就是加不上, 然後沒時間弄 UI, 所以很醜 XD

然後想說順便測一下 djangy 跑起來如何, 就把 instawall deploy 到 djangy 上去了, 跑看看如何, 目前 djangy 用起來挺順的, 透過 git deploy 實在是很方便。

不過寫到下午 3 ~ 4 點的時候, 突然覺得寫小玩具好浪費時間, 因為最近有很多書要讀, 似乎看書還比較符合成本效益 ~”~ 所以 … 就先擺著了, 反正就花幾個小時而已, 就當是測試吧!

不過有一點心得, 本來是想做成 map 的, 沒想到撈了一下 popular, 居然一百張裡面只有兩張有 geo 資料 …
