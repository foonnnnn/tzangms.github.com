Stickeraction 主機搬到 Prgmr

Aug 9th, 2009

其實 prgmr 已經買了一個多月了, 可是最近超懶, 提不起勁幫 stickeraction 搬家, 剛好由於這幾天 stickeraction 一直當機, 所以乾脆提起勁來把 stickeraction 從 linode 搬到 prgmr。

其實最主要是想省點錢, 因為 linode 每個月就要花掉我大約一千塊的台幣, 可能也是因為這大半年來幾乎都沒在改 stickeraction 的東西, 就一直擺在那邊, 所以感覺有點浪費錢的感覺 XD

而目前在 prgmr 買的主機是一台 512MB RAM, 不過只有一顆 CPU, 跑起來感覺滿”ㄍㄧㄥ”的 XD

Linode ($29, 540MB, 4CPU)

vendor_id    : GenuineIntel
cpu family  : 6
model       : 23
model name  : Intel(R) Xeon(R) CPU           L5420  @ 2.50GHz
stepping    : 6
cpu MHz     : 2500.086
cache size  : 64 KB

Prgmr ($12, 512MB, 1CPU)

vendor_id    : AuthenticAMD
cpu family  : 16
model       : 4
model name  : Quad-Core AMD Opteron(tm) Processor 2374 HE
stepping    : 2
cpu MHz     : 2200.000
cache size  : 512 KB

4 顆 xeon 比起 1 顆 Quad-Core 不知道哪個比較夠力? anyway, 省點錢用 ~”~

這幾天的不穩定大家請多包含了 Orz
