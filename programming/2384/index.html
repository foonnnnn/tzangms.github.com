Easy Unfollow 重出江湖!

Mar 13th, 2009

去年在參加完 Google Developer Day 2008 之後, 因為當時 twitter 很慢, 所以寫了 easy unfollow 這個工具, 希望大家能夠 unfollow 你親愛的推友, 來救 twitter, anyway 不過當時不會 python, 所以 bug 太多也就沒有正式 release 了 XD

由於最近在開始學 python, 然後明天要去吃哈跟大使, 所以稍微練習一下, 把這個 easy unfollow 這個工具重新寫出來 ( 因為之前的 source code 都不見了 XD ), 複習一下 Google App Engine, 也體驗一下令人驚豔的 GAE Framework: GAEO。

不過還是有 bug, 目前知道的是在 session expire 時好像會有錯誤 XD

有興趣, 想要 unfollow 你在 twitter 上面的好友的話, 歡迎試用! :D 請到 http://easyunfollow.appspot.com
