最近在學 Python 跟 Django

Mar 27th, 2009

最近在開始學 python 跟 django, 也將用在新工作上, 所以由於不是很熟, 所以也伴隨著壓力, 畢竟不像以往用 php 用的很得心應手。 但是”用久了就是你的了”, 沒錯吧! 而我所以最近看完了 django book, 在 k django 的文件, 還有好多文件要 k, 得要先把文件都 k 完才不會浪費時間去重造輪子。 等 django 讀完, 再來去好好研究 python。

而在研究 django 這一些時間, 都會覺得自己以前在用 php + codeigniter 寫程式的時候真的是浪費很多時間, 然而應該是把點子實現出來, 才是重點吧。 程式再怎麼會寫, 沒有點子也不知要寫啥, 不過這件事情的看法當然是因人而異。

我覺得 django 可以讓我的生活更簡單一些。

而過去一年, 除了工作, 還有加強自己的程式設計的能力之外, 就是花了很多時間在專輯的錄音、後製上, 雖然有些地方沒有弄的很好, 不過後來收到一些關於音樂錄製的 feedback, 感謝各位。 不過在去年一整年的時間, 我卻連一首歌都沒有寫出來。 記得從大三開始, 每年都至少會寫出 3 ~ 4 首歌, 即使當兵的時候因為週休, 所以仍然一直在進行樂團表演, 當兵一年半也有寫出幾首歌曲。 而第一份工作那兩三年, 也寫了 10 幾首歌。 但是去年卻一整年沒有寫出一首歌, 似乎我把很多時間都花在寫程式, 研究技術上面了。

我想最近新工作安定下來後, django 跟 python 也熟悉之後, 是該來開始寫歌了。

最後, 我越來越喜歡寫漸進式離題的文了
