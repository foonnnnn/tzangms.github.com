梁朝偉 XD

Jun 20th, 2007

話阿潑真的是像極了梁朝偉, 怎麼樣的一個像法? 有圖有真相, 請看這篇, 這就難怪了, 每次一到公司, 就有一股…一股像是在演電影的感覺。

其實這件事是在不久前, 阿潑在 twitter 上提到, 有人說她像梁朝偉, 這一提, 真是一語驚醒夢中人阿!! 以前的記憶就這樣往我衝了過來, 沒錯! 阿潑真的是像極了梁朝偉啦!! 超像的啦!! 我的天阿! (讚嘆) 我猶豫了好久~ 不知道要不要寫這一篇, 呵, 最後我還是要來讚嘆一下這件事情! 阿潑太帥啦!! XD

話說先前 “無間道之 CD PRO II” 正紅的時候, 吉他手小凱給了我 “肚爛明” 這個外號 XD (我的名字裡有個”明”字… ) 我也給了小凱 “GY仁” 的稱號 (也很剛好的, 小凱的名字裡有個”仁”字), 就這樣的, 在過去那段美好的時光, 我和小凱總是演起 CD PRO II 的橋段。

在我的大學時代的回憶中, CD PRO II 真的是佔了很重要的一部份, 可是翻遍了電腦, 卻不見那埋藏已久的回憶, 噗, CD PRO II 的影片都不見啦!!! Orz 幸好, 現在有 youtube, 沒錯, 就請大家跟我回憶一下 CD PRO II 吧!! 我都快哭了~~

GY 仁 !! 肚爛明 !!
