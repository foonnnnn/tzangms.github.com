找到房子了~ Murmur

Jul 19th, 2007

在上禮拜六終於找到房子了, 所以前兩三個禮拜每天都煩惱著找房子的事情, 已經不存在了, 倒是最近我 blog 的更新次數減少了, 最主要的原因就是有事要忙, 忙著搬家的一堆事, 網路、帳單…等等的, 還要整理東西, 加上這禮拜又值班, 有些事情也力不從心, 還有最主要的原因也是因為我的左手, 一直呈現著無力狀態, 我的脖子一直都很痛, 這應該算是職業病吧? 有人說我是姿勢不良, 但是公司每個人的桌子都一樣高, 雖然我是一直覺得桌子太高, 但是我又不想把椅子調太高, 因為會跟對面的人目光直視 XD anyway, 反正我的手、肩膀跟脖子一直都很難過就是了, 所以越來越懶得打 blog。

解決的辦法呢?! 就是去看醫生吧! XD

其實現在如果可以的話, 我真的希望我的左手可以不要動… 這種情況應該已經持續了有一年之久, 上班沒多久就開始有這種狀況 …

所以身體一直不舒服的狀況, 讓我有種倦怠感… 加上昨天值班又出包, 整個心情就很糟糕, 然後就有產生了心有餘而力不足的感覺。由於身為網路程式設計師, 要學的東西真的很多, 除了目前已經算熟的 PHP 之外, 還要會 javascript 會 AJAX, 但另一方面還要熟 database, 但對於 database 方面, 我算是有點弱, 都是在工作才慢慢自己在學習, 但是~ 然而目前在打算考 Zend PHP Certification 的狀況下, 其實也沒額外的心力在去讀 SQL 的東西, 只能慢慢來, 更別說 Server 管理之類的事情了, 不過朋友說, 我還年輕… 靠, 就是不想比別人弱阿 XD

然後, 先前在公司架好的 SVN server 其實也在等著我更進一步的了解, 想弄到 Auto deployment 這個地步, 也就是要有 development, stage, 跟 production 這三個 enviroment。

但是目前在公司進行某項工作, 所以也沒有太多時間去摸索更多 SVN 的東西, 目前 SVN 在公司只發揮到 source revision control 而已, 但我現在只能快把手上的工作快點作好, 不然會拖到 myshare 的新功能, 我一直很想把 myshare 的新功能作出來, 否則等到時間過了, 或者是某項新東西出來了, 這個功能可能就過時了, 更慘的是會變成 copycat。

不過最好還是在這些改頭換面的新東西出來前, 把 Auto deployment 弄出來, 這樣才有 web 2.0 公司專業的感覺阿!! 當然這也是要當人家問起才會有人知道 XD 然而要把東西作大, 還要快點把 Building Scalable Web Sites 給 K 完。

總和以上, 我目前需要 k 的東西真的滿多的

  • Zend PHP Certification
  • Subversion
  • Building Scalable Web Sites
  • Learning SQL
  • Learning jQuery

衝阿!! 阿 跌倒了.. XD

呵 (苦笑) 你好, 我很弱…
