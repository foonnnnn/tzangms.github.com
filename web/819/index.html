怕貼圖連結失效? 試試 imgred.com

Apr 7th, 2007

通常在論壇或是BBS在貼文時, 會貼上某些圖片的連結, 最直接的方法就是直接貼上圖片的連結網址, 但是當某天那個網站關了, 或者是圖片被刪掉, 文章裡頭的圖片就變成了一個叉燒包, 所以可能進行下列的步驟:

  1. 會把想要的圖片抓下來存檔
  2. 然後放到自己的空間(或 Flickr?)
  3. 用自己已經上傳自己空間的圖片網址, 貼到論壇或BBS
  4. 刪除自己電腦上剛存下來的圖片

這未免也太吃力了, 現在可以透過 imgred.com 來輕鬆進行最直接的貼圖方式。只要將圖片網址前面加上下面這行即可:

http://imgred.com/

嗯, 就是這麼簡單, 只要加上這個網址就不用在怕貼到論壇或是BBS的圖片變叉燒包了! 嗯, 用 firefox 的圖來做例子好了:

原始圖片連結:

http://www.mozilla.com/img/firefox-title.png

用 imgred.com 來轉址

http://imgred.com/http://www.mozilla.com/img/firefox-title.png

另外還會產生縮圖!! 縮圖的網址如下, 加上 tn/ 即可

http://imgred.com/tn/http://www.mozilla.com/img/firefox-title.png

PS. 實際測試的結果, 好像太小的圖在產生縮圖會有問題, 這張圖縮圖就掛點, 不過原圖是 ok 的 XD

沒錯, 就是這麼簡單, 只要加上 http://imgred.com/ 就可以了! 根據 imgred.com 的說法, imgred.com 會在第一次收到這個轉址的時候, 將你附在後面的網址圖片抓到 imgred.com 的空間上, 之後就會一直存在 imgred.com 的空間了, 所以即使原來的 firefox 圖片被刪掉了, 或是網站掛點, 經過 imgred.com 轉址所貼到論壇或 BBS 的圖片連結都不會失效, 方便又省力! (不過… 其實如果 imgred.com 本身掛了, 那也沒輒 XD)

不過剛剛去 imgred.com 看了一下, 不知道他要靠什麼存活? XD anyway, 總是一個方便的服務, 希望他能好好活下去囉 XD
