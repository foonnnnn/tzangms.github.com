Shareaholic 使用 Twitthat !?

May 23rd, 2008

有時候比較空閒或是無聊的時候, 我都會到 twitthat 的 most read 頁面, 看看有沒有哪些文章被比較多推友閱讀過, 如果有看到感興趣的標題, 就會點來看看。 (另外也順便看看 twitthat 有沒有當掉 XD) 不過最近這幾天使用 twitthat 的外國推友好像突然暴增, 我看 referral 也看不出來是哪邊連過來的。

後來又一直在 most read 看到 shareaholic 這個網站一直被推上來, 有點稍微覺得有點好奇, 難得有非中文的文章獲得那麼多的推(因為中文使用者居多嘛 XD), 只是我都沒有去看看 XD 後來今天下班前, 推友 quanmengli 跟我說 :

quanmengli @tzangms 难道偶眼花了吗？twitthat替代twitthis成为shareaholic首选？

我驚訝了一下, 馬上跑去裝了 shareaholic 的 firefox extention。 不過裝完後, 只有看到 twitter this 這個選項, 沒想到按下去之後, 開出來的視窗居然是顯示 twitthat!! 當時真的是有嚇一跳! hmm… 看來應該是因為先前 twitthis 掛太久才會換成 twitthat。

Anyway, 使用 Firefox 的推友, 除了 twitthat 的 bookmarklet 之外, 以後還可以直接裝 Shareaholic 這個外掛, 透過這個外掛來 twitthat 啦! :D
