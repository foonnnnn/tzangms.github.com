identi.ca - PHP Open Source Twitter?

Jul 3rd, 2008

identi.ca, 一個新的微網誌服務, 其實現在微網誌服務真的是很多了, 不過這個比較特別一點的是, identi.ca 用 PHP 開發, 可以用 jabber/GTalk, 也可以用 openid 來登入, 而且是 open source!!?

不過目前為止連到這個網址 http://identi.ca/doc/source, 上面是這樣寫的:

You can get a copy of the software from the Laconica main site. The version of the software that runs on this site is unmodified from that version. The site also depends on certain libraries and other software; you can get those at the Laconica site, too.

不過找了一下, 還沒看到任何 source code 的蹤影 :p 不過那不重要, 因為…可以離開 twitter 就很厲害了 XD
