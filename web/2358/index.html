Stickeraction Clone?

Feb 15th, 2009

平常沒事就會逛逛 stickeraction, 看看有沒有什麼新貼紙, 不過前幾天, 在參加完 twestival 回家之後, 看了一些新的貼紙, 點進某張貼紙發起人的網站, 就發現了這個網站了, 叫做貼貼王, 好像現在網站都要叫 “XX王” 之類的 XD

沒想到還有人願意 clone stickeraction, 連網址都差不多 ( actionstick.com ), 就反過來而已。 不過~ 其實不管是 layout 或文案很像, 我是覺得都沒差, 畢竟網站不就都是這樣, 有同樣目的, 文案當然也會差不多。 只是… stick 的名詞似乎是棒子 (?)

Anyway, 我比較 care 的是, 怎麼連好幾張在 stickeraction 的串連貼紙都抄過去? 我基本上是直覺, 被拿過去的貼紙並沒有經過發起人, 或是貼紙作者本身的同意。

另外, 這個站似乎都是全手動的, 發起串連要人工, 加入串連也要人工, 這樣不會很累嗎? anyway, 我也不知道製作這個網站的公司是在想啥, 老闆們的想法不是我能夠理解的, 不過貴公司有人看到的話, 倒是可以考慮一下來跟 stickeraction 合作阿? 我可以弄個 sub domain 給你們, 然後 css 自定 layout 之類的 XDD

直接合作的話, 會比請工程師來作一個串連平台, 省力又省錢呦!! XD 最後也感謝讓我 blog 有題材可以寫, 最近都不知道寫啥 Orz
