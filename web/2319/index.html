Twitthat Firefox Extension!

Jan 13th, 2009 | Comments

[caption id=”attachment_2322” align=”aligncenter” width=”580” caption=”you can access twitthat on the right click menu”][/caption] 感謝 kej 大大幫 twitthat 做了個 firefox 的外掛! 大家可以到 twitthat 的這個頁面點 Click here to install 進行安裝, 或是到 TwitThat :: Firefox 附加元件 這個頁面進行安裝 ( 需要帳號 )

裝完這個外掛之後, 就可以直接按右鍵就可以在 firefox 的右鍵選單看到 twitthat 的選項囉! 另外, 也可以像下面這張圖一樣, 把 twitthat 的按鈕拉到 toolbar 上呦!! :D [caption id=”attachment_2323” align=”aligncenter” width=”580” caption=”you can add twitthat button on the toolbar”][/caption]

不錯吧! kej 真是太酷了!! 有用 Firefox 的朋友趕快安裝這個外掛吧!!! 到這個頁面點 Click here to install 安裝吧!!
