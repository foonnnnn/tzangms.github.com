Imified 五分鐘建立自己的機器人

Sep 10th, 2008

今天看到 imified 就玩了一下, 真的是很神奇又方便阿!! imified 可以讓你建立一支機器人, 接著你只需要指定一個程式的網址, 就可以讓機器人開始運作了! 而且整個流程超簡單的 :p

在建立好機器人之後, 會讓你指定一個程式的網址, 接下來就是透過這支程式, 接收 imified 機器人的輸入, 而程式只要輸出(印出)訊息, 就會直接透過機器人發訊回覆。 不過目前申請的機器人好像就只有 Jabber/Gtalk(?)

用 PHP 做個簡單範例: [source:php] <?php // 接收 im 訊息 $msg = $_REQUEST[‘msg’];

// 輸出訊息到 im echo $msg; ?> [/source]

就是這樣, 非常簡單阿 XD

所以一開始想到的, 就是寫一個可以送訊息到 twitter 的機器人, 不過由於帳號是寫死的, 就不公佈出來了 XD 然後在查 Yahoo! 奇摩字典的時候, 又突然想到, 乾脆寫個查單字的機器人好了, 由於抓 Yahoo! 奇摩字典的程式也是我之前就寫好的一個 library, 所以真的包含註冊帳號、設定跟寫程式的部份, 真的只花了差不多五分鐘就弄好了 :p

有興趣的朋友可以用 Gtalk/jabber 加入下面這個機器人試看看

chinese_dict@bot.im

只能輸入英文單字喔 :p 而這支程式的程式碼大概像下面這樣 [source:php] <?php require ‘yahoo_dict.php’;

$msg = $_REQUEST[‘msg’]; $word = trim($msg);

if (!preg_match(‘/\w+$/’, $word)) {

echo '請輸入英文單字, 以進行單字查詢';
exit;

}

$yahoo_dict = new yahoo_dict; $explain = $yahoo_dict->get_explain($word);

echo ($explain)?implode(“\n”, $explain):’查無此單字’; ?> [/source] 真的是很方便快速阿, 整個流程簡化成這樣了, 有興趣的人趕快去玩, 寫些東西來玩玩吧!? XD

imified 把整個機器人的動作都簡化了, 超方便的阿!! 有興趣玩看看的朋友, 只要到 這一頁 去 request, 接下來的流程也很順, 一下子就可以啟用機器人了 :D

然後看看他的 API Document 可以更清楚知道 imified 的機器人提供哪些參數可以用。

不過目前看到在 userkey 的部份, 本來以為是 im 帳號, 沒想到是編過的, 所以即使拿到這個 userkey 也沒辦法知道是誰用了機器人, 好像沒辦法直接跟 web 界面接起來? 也許只能做 in box 的應用? anyway, 有空在研究看看, 還滿好玩的就是了 XD
