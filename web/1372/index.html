黑特那! Hatethat!

Aug 6th, 2008

由於黑特銀行的出現, 覺得未來所謂 twitter like 的微網誌, 在台灣將會有所謂分眾性的可能發生 (誤) 所以, 深深覺得不只有幹可以在黑特銀行說, 對網站或文章不爽, 也該有個 bookmarklet 可以直接送到黑特銀行阿!!!

所以黑特那!就這麼誕生了 (囧) 基本上是直接拿 twitthat 來改, 所以今天晚上一下子就寫好了 XD 不過要登入黑特那! 則是要先取得 api key, 所以請先登入黑特銀行之後, 點選帳戶資訊, 看到最下方就可以看到 api key 啦!

另外… 下面開放建議黑特那!的動作, 把他加進黑特那的選項, 目前有… 1. 幹! 這什麼爛東西 2. 幹! 這什麼爛文章 3. 幹! 這什麼爛圖片 4. 幹! 我快笑死了 5. 幹! 我太失望了

請大家踴躍發言 XD ( 噗, 突然覺得沒人會理我 ~”~ )
